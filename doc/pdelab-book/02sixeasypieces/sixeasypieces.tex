\begin{abstract}
This article is the first of  series that 
explains how discretization schemes for partial
differential equations can be built in a generic way using the 
``Distributed and Unified Numerics Environment (DUNE)'', see {P. Bastian, M. Blatt,
A. Dedner, C. Engwer, R. Klöfkorn, M. Ohlberger, O. Sander:} {\itshape
  A Generic
Grid Interface for Parallel and Adaptive Scientific Computing. Part I:
Abstract Framework.} Accepted for publication in  Computing. 

Here we
show the construction of finite element function spaces and treat six
different discretization schemes for solving the stationary diffusion
equation. 
\end{abstract}
